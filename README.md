# ArcSolis marketing site

This repository contains the ArcSolis marketing website built as a static HTML, CSS and JavaScript experience.

## Local development

Open `index.html` directly in your browser or serve the folder with any static file server.

## Structure

- `index.html` main page markup.
- `styles.css` layout and theme, using ArcSolis navy and solar orange tones and avoiding purple gradients.
- `script.js` small enhancements for navigation, smooth scrolling and FAQ interactions.
